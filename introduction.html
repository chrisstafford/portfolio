Introduction

Hello.

You're probably wondering what you're looking at. What the goal of this small site is, is to exhibit my portfolio while demonstrating some of the abilities on my resume. I've written the code for this page to resemble the practices used to build a web application (To the best of my ability when applying these practices to build a small portfolio site, of course.)

I invite you, at any point, to look at the source files listed below if you would like to take a better look at my work. While I can't promise this will be the most entertaining portfolio, I believe you will get a good understanding of the caliber of my work.

One of the first things you'll notice, I hope, is that there is not a lot of code. I believe that the less code there is, the less that can go wrong, and also, the less amount of effort it takes to correct an issue should one occur. For example, if you were to right click and view source on this page, you'd notice that there is only 28 lines of code. I would also like to point out that this page functions and displays correctly in all current versions of web browsers and mobile browsers.

HTML Files

  • index.html - This page file as the template in which all the other files are loaded into.
  • introduction.html - This is the file containing the content you are currently reading.
  • portfolio.html - This file contains the template in which the portfolio items are loaded into via a JSON recordset.
  • resume.html - This file contains the contents of my resume.

CSS Files

  • default.css - This file contains the general layout and font styles used throughout this site.
  • portfolio.css - This file contains only styles relative to portfolio.html

Javascript Files

  • jquery-1.10.2.min.js - jQuery v 1.10.2
  • default.js - This file initializes most of the functionality within the site.
  • data.js - This file contains the JSON default.js uses to pull data from.
